QUICK INSTALL (3 Steps):
========================

1. EXTRACT THE ZIP FILE
   - Right-click "anvil-solo-setup.zip"
   - Click "Extract All..."
   - Choose a location (e.g., C:\Program Files\Anvil Solo)
   - Click "Extract"

2. RUN THE APP
   - Open the extracted folder
   - Find "Anvil Solo.exe"
   - Double-click to run
   
3. CREATE SHORTCUT (Optional)
   - Right-click "Anvil Solo.exe"
   - Click "Create shortcut"
   - Drag shortcut to Desktop

FIRST TIME SETUP:
=================

When you launch Anvil Solo:

1. Create or Import Wallet
   - Choose "Create New Wallet" or "Import Existing"
   - Set a strong password
   - SAVE YOUR SEED PHRASE (write it down!)

2. Fund Your Wallet
   - Copy your wallet address
   - Send SOL from an exchange
   - Wait for confirmation

3. Start Trading
   - Create a strategy (DCA, Ratio, or Bundle)
   - Configure your settings
   - Click "Start"
