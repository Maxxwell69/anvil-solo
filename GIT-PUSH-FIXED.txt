OPTION 1 - Test Locally (Quick):

OPTION 2 - Cloudflare R2 (Best for Production):
-----------------------------------------------
1. Go to Cloudflare dashboard
2. Create R2 bucket called "anvil-downloads"
3. Upload anvil-solo-setup.zip to bucket
4. Make it public
5. Get the URL (e.g., https://pub-xxxxx.r2.dev/anvil-solo-setup.zip)
6. Update download route to redirect to that URL

Free, fast, no limits!

OPTION 3 - Railway CLI (Direct Upload):
---------------------------------------
npm install -g @railway/cli
railway login
cd cloud-services
railway link
railway run "cp ../anvil-solo/anvil-solo-portable.zip public/downloads/anvil-solo-setup.zip"
